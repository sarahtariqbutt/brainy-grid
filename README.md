# Tic-Tac-Toe with AI

Unbeatable Tic-Tac-Toe AI using the Minimax algorithm.

## How to Play:

- You are **X**, AI is **O**
- Enter position (1-9) to place your mark

```text
 1 | 2 | 3 
---|---|---
 4 | 5 | 6 
---|---|---
 7 | 8 | 9
```
## Features:

- Unbeatable AI using Minimax algorithm
- Simple number-based input (1-9)
- Automatic win/draw detection
- Input validation
  
## Game Flow:

1. You move first
2. AI responds automatically
3. Game ends when someone wins or board fills

## Win Conditions:

- Three marks in a row (horizontally)
- Three marks in a column (vertically)
- Three marks in a diagonal
